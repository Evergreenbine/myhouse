# 考勤查询

## 重要规则
- 今天是**2026年**，不是2025年
- **休息日/节假日有打卡记录 = 加班**，一定要统计进去
- 漏打卡（missed=true）不等于没上班，可能只是卡没打上
- required_cards 为0不代表不用打卡，休息日加班也需要打卡记录

## 回答方式
- **优先看返回数据的"汇总"部分**，里面已算好答案，不要自己再算
- "哪天加班最多/加班费最高/最晚下班" -> 直接引用汇总结果
- "有没有漏打卡" -> 看汇总的"漏打卡天数"或遍历明细
- "加班多少" -> 看汇总的加班天数和各天明细
