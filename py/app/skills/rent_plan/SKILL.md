# 收租计划 Skill

## 适用问题

用于回答和检查每月收租计划进度，包括账单录入、待发送、待收款、已收款和异常状态。

常见问法：
- 本月谁还没有录入账单
- 这个月哪些房间待发送账单
- 谁待收款，谁已经收款
- 101 房间现在走到哪一步了
- 帮我检查本月收租计划有没有异常

## 可用数据

- 有效合同列表：房间、楼栋、租客、月租、水电单价
- 指定账期账单：账单状态、应收金额、房租、水费、电费、其他费用
- 收款记录：已收金额、收款日期、收款方式

## 状态解释

- 未录入：有效合同存在，但指定月份没有账单
- 录入中：账单状态为 draft
- 待发送：账单状态为 pending
- 待收款：账单状态为 pending_payment 或 unpaid
- 部分收款：账单状态为 partial，或已收金额小于应收金额
- 已收款：账单状态为 paid，或已收金额不小于应收金额

## 能执行的动作

- 查询指定月份所有房间的收租进度
- 查询指定月份未录入账单的房间
- 查询某个房间的账单进度解释
- 检查收租计划异常，例如缺账单、金额为零、已收大于应收、状态和收款不一致

## 禁止动作

- 不直接新增、修改、删除账单
- 不直接发送账单
- 不直接确认收款
- 不替用户终止合同或改租客状态

## 回答要求

- 优先给出房间号、租客、状态、应收、已收、待收
- 如果用户没有指定月份，默认使用当前月份
- 如果数据缺失，要明确说明缺少哪类数据
- 金额使用人民币，两位小数
