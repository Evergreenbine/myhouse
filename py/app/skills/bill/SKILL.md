# 账单 Skill

## 适用问题

用于解释账单金额、查询账单构成、生成账单草稿和检查账单预览数据。

常见问法：
- 这个账单为什么是这个金额
- 101 本月账单构成是什么
- 帮我生成 101 本月账单草稿
- 输出 101 本月账单图片
- 生成这个账单的收据图片
- 检查账单预览数据有没有问题
- 水费和电费怎么算出来的

## 可用数据

- 合同：月租、水费单价、电费单价、租客、房间
- 账单：房租、水费、电费、其他费用明细、其他费用汇总、总金额、状态
- 水电读数：上期读数、本期读数、用量
- 收款记录：用于解释账单是否已经收款或部分收款

## 计算规则

- 总金额 = 房租 + 水费 + 电费 + 各项其他费用之和
- 水费 = 水表用量 * 合同水费单价
- 电费 = 电表用量 * 合同电费单价
- 用量 = 本期读数 - 上期读数

## 能执行的动作

- 根据账单 ID 解释金额
- 根据房间和月份查询账单构成
- 生成账单草稿数据，但不保存
- 检查账单预览是否缺少合同、读数、金额或照片
- 返回 AIChat 渲染账单图片所需的数据
- 当用户明确要求生成账单，且读数数据齐全时，先准备账单草稿和账单图片，用户确认后再保存
- 支持用户通过对话修改账单金额、读数或其他费用；网费、卫生费等其他费用必须按“项目名称 + 金额”逐项记录
- 用户明确要求覆盖已有账单或旧账单图片时，准备覆盖确认事项；确认后更新原账单并返回最新账单图片

## 禁止动作

- 不在缺少必要读数时保存账单
- 不静默覆盖已有账单，覆盖必须由用户明确说明并确认
- 不直接更新账单状态
- 不直接确认发送

## 回答要求

- 金额使用人民币，两位小数
- 读数只返回数字，不附带单位
- 草稿必须明确标注“未保存”
- 发现缺读数、缺合同、金额为负数时要提示人工确认
